# TBH Tracker

Tracker de performance para o jogo **Task Bar Hero**.

## Funcionalidades
- Registre seus tempos de farm
- Compare builds de personagens e skills
- Calculadora de Farm EXP com penalidade de nível

## Como baixar
Acesse a aba [Releases](https://github.com/th3dest/TBH-Tracker/releases) e baixe o `.exe` mais recente.

Não precisa instalar nada — é só executar.
